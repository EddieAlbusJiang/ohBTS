# RunningLambda

蓝牙串口助手 —— 基于 HarmonyOS NEXT 的蓝牙 SPP（Serial Port Profile）终端应用，可通过无线蓝牙连接与串口设备（如 HC-05/HC-06 模块、嵌入式系统、IoT 设备等）进行数据收发，方便调试过程。

## 功能概览

- **蓝牙 SPP 设备扫描与连接** —— 扫描附近蓝牙设备，可按设备名过滤，一键建立 SPP 连接，实时显示连接状态
- **快捷命令管理** —— 创建、编辑、删除常用指令；支持 JSON 格式导入/导出，跨设备复用
- **重复发送与自定义间隔** —— 为快捷命令设置重复发送次数，并在设置中配置自动重发间隔（毫秒），实现批量/自动化操作
- **常规串口自由输入** —— 提供自由输入的串口终端界面，可随时发送任意内容
- **ASCII / HEX 双模式** —— 收发数据均支持 ASCII 文本与十六进制两种显示和输入模式
- **实时收发日志** —— 终端风格的数据展示区域，发送/接收记录以颜色区分，附带毫秒级时间戳，自动滚动至最新数据

## 权限说明

应用运行需授予以下权限：

- 蓝牙相关权限（使用、访问、发现设备）
- 位置权限（Android/HarmonyOS 蓝牙扫描所需）
